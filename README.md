## Fitur Unggulan

- 🛒 **Point of Sale (POS)**
  - Antarmuka kasir cepat dengan pencarian produk real-time
  - Scan barcode produk
  - Multi-payment: **Tunai**, **QRIS**, **Midtrans** (QRIS, E-Wallet, Virtual Account)
  - Kalkulasi PPN (pajak) otomatis sesuai pengaturan toko (default 11%)
  - Diskon per item dan per transaksi
  - Cetak struk ke printer thermal (ESC/POS) atau PDF
  - Nomor resi otomatis

- 📦 **Manajemen Produk & Inventaris**
  - Katalog produk dengan kategori dan satuan unit
  - Kode produk otomatis (auto-generate)
  - Barcode unik per produk
  - Harga beli & jual, stok minimal (alert stok menipis)
  - Upload & optimasi gambar produk otomatis
  - Tracking riwayat stok (Stock Movement Log): barang masuk, keluar, penyesuaian, retur
  - Filter produk aktif/non-aktif

- 👥 **Manajemen Pengguna & RBAC**
  - Role-based access: **Owner** dan **Kasir**
  - CRUD user dengan toggle aktif/non-aktif
  - Reset password dari panel admin
  - Autentikasi penuh: Login, Register, Lupa Password, Verifikasi Email, **2FA (TOTP)**

- 💸 **Manajemen Keuangan**
  - Pencatatan pengeluaran operasional harian (kategorisasi)
  - Laporan Penjualan (filter tanggal, export Excel)
  - Laporan Pengeluaran (filter tanggal, export Excel)
  - Laporan Laba Rugi otomatis (Omzet - Pengeluaran)
  - Margin keuntungan dalam persentase

- 📊 **Dashboard & Analitik**
  - Ringkasan penjualan hari ini (jumlah transaksi, revenue, item terjual)
  - Produk terlaris minggu ini
  - Alert stok menipis
  - Chart penjualan real-time (per jam, per hari)
  - Statistik terupdate dengan lazy loading

- 🏪 **Pengaturan Toko**
  - Nama toko, alamat, email, telepon, NPWP
  - Upload logo toko & gambar QRIS
  - Konfigurasi Midtrans (merchant ID, client key, server key)
  - Pengaturan tarif pajak (PPN)
  - Footer struk & ukuran kertas printer

- 🤖 **AI Business Consultant (POS Agent)**
  - Chatbot cerdas berbasis **Google Gemini 2.5 Flash**
  - 8 tools business intelligence: penjualan hari ini, produk terlaris, stok menipis, penjualan periode, kategori, pengeluaran, laba rugi, produk tersedia
  - Multi-provider siap pakai: **Gemini**, **Anthropic Claude**, **OpenAI GPT**
  - Riwayat percakapan & tracking penggunaan tool
  - Statistik chat (total percakapan, pesan, tool usage)
  - Hanya untuk Owner, terintegrasi di sidebar

- 💳 **Midtrans Payment Gateway**
  - Integrasi penuh: QRIS, E-Wallet (GoPay, OVO, Dana, dll), Virtual Account
  - Webhook notification handler untuk sinkronisasi status otomatis
  - Redirect handler untuk finish payment flow
  - Mode sandbox & production (via pengaturan toko)

- 🖨️ **Cetak Struk**
  - Printer thermal langsung (ESC/POS protocol via USB/network)
  - PDF receipt untuk simpan digital
  - Footer struk yang dapat dikustomisasi

- 🔍 **SEO**
  - SEO settings per halaman (title, description, keywords, OG image)
  - Sitemap.xml otomatis

- 🎨 **Tampilan**
  - Dark mode / Light mode (next-themes)
  - UI modern dengan shadcn/ui & Tailwind CSS v4
  - Layout responsif dengan sidebar navigasi

## Setup Gemini AI API Key (Wajib untuk Fitur Chatbot)

Fitur AI Business Consultant membutuhkan API Key dari layanan LLM. Berikut cara mendapatkannya:

### Google AI Studio (Gemini) — Default

1. Buka [Google AI Studio](https://aistudio.google.com/app/apikey)
2. Klik **"Get API Key"** → **"Create API Key"**
3. Pilih project Google Cloud atau buat baru
4. Salin API Key yang dihasilkan (diawali `AIza...`)
5. Edit file `.env`:
```bash
GEMINI_API_KEY=AIzaSy...apikey-anda-disini
```

### Alternatif AI Model Lain

Selain Gemini, aplikasi juga mendukung provider lain melalui Prism PHP. Cukup isi API Key provider yang diinginkan di `.env`:

**Anthropic Claude:**
```bash
ANTHROPIC_API_KEY=sk-ant-...apikey-anda
```

**OpenAI GPT:**
```bash
OPENAI_API_KEY=sk-...apikey-anda
```

**OpenRouter (Multi-model):**
```bash
OPENROUTER_API_KEY=sk-or-...apikey-anda
OPENROUTER_URL=https://openrouter.ai/api/v1
```

### Ubah Provider Default (Opsional)

Di `config/prism.php`, ganti `default_driver` ke provider yang diinginkan:
```php
'default_driver' => Provider::Gemini,   // atau Provider::Anthropic, Provider::OpenAI
```

## Midtrans Setup

1. Daftar akun [Midtrans](https://midtrans.com)
2. Dapatkan **Merchant ID**, **Client Key**, **Server Key** dari dashboard Midtrans
3. Isi di `.env`:
```bash
MIDTRANTIS_CLIENT_KEY=
MIDTRANS_SERVER_KEY=
MIDTRANS_MERCHANT_ID=
MIDTRANS_IS_PRODUCTION=false
```
4. Atau isi melalui halaman **Shop Settings** di aplikasi
5. Untuk testing lokal, gunakan **Ngrok** (lihat petunjuk di bagian Testing)

## Printer Thermal Setup

```bash
# Di .env, atur koneksi printer:
PRINTER_CONNECTION_TYPE=network   # network / windows / file
PRINTER_NETWORK_IP=192.168.1.100
PRINTER_NETWORK_PORT=9100
PRINTER_WINDOWS_PATH=LPT1
PRINTER_FILE_PATH=/dev/usb/lp0
```

## 🚀 Instalasi Cepat

### 1. Clone & Install
```bash
git clone <repo-url>
cd <directory>

composer install
npm install
```

### 2. Environment Setup
```bash
cp .env.example .env
php artisan key:generate
```
Edit file `.env` dan sesuaikan konfigurasi database, Gemini API Key, dan Midtrans.

### 3. Storage Setup
```bash
php artisan storage:link
```

### 4. Database Migration & Seeding
```bash
php artisan migrate --seed
```

### 5. Jalankan Aplikasi
```bash
composer run dev
```

## Route Utama

Sistem menggunakan **RBAC** dengan role **Owner** dan **Kasir**.

### Management (Owner Only)
| Route | Fitur |
|---|---|
| `GET /users` | Manajemen User & Staff |
| `GET /product-categories` | Kategori Produk |
| `GET /units` | Satuan Unit Produk |
| `GET /products` | Katalog Produk & Stok |
| `GET /stock-movements` | Riwayat Mutasi Stok |
| `GET /expense-categories` | Kategori Pengeluaran |
| `GET /expenses` | Pencatatan Pengeluaran |
| `GET /shop-settings` | Pengaturan Toko & Pajak |
| `GET /reports/sales` | Laporan Penjualan |
| `GET /reports/expenses` | Laporan Pengeluaran |
| `GET /reports/profit-loss` | Laporan Laba Rugi |
| `GET /settings/profile` | Edit Profil |
| `GET /settings/security` | Ubah Password |

### Operational (Owner & Kasir)
| Route | Fitur |
|---|---|
| `GET /pos` | Antarmuka Kasir (Penjualan) |

| `GET /chat` | AI Business Consultant |
| `GET /dashboard` | Dashboard Statistik |
| `GET /settings/appearance` | Tema (Dark/Light) |

| Route | Fitur |
|---|---|

| `GET /api/dashboard/charts` | Data Chart Dashboard |
| `GET /api/dashboard/hourly` | Data Penjualan Per Jam |
| `GET /api/dashboard/charts/refresh` | Refresh Statistik |

## 📡 Testing Pembayaran (Local with Ngrok)

1. **Jalankan Ngrok:** `ngrok http 8000`
2. **Update APP_URL:** Salin URL Ngrok ke `.env` (misal: `APP_URL=https://abcd.ngrok-free.app`)
3. **Konfigurasi Midtrans:** Isi **Payment Notification URL** di Dashboard Midtrans dengan `https://abcd.ngrok-free.app/midtrans/notification`
4. **Test:** Lakukan transaksi di halaman POS via Midtrans dan selesaikan pembayaran di simulator
